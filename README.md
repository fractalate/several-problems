## Help

Please see [MANIFEST.md](./MANIFEST.md) for an outline of the files in this project.

### Languages

**Go**

```bash
go run PE-001.go
```

**Julia**

In the `julia` interpreter:


```julia
using Pkg
Pkg.add("Primes")
```

then

```bash
julia PE-001.jl
```

**Python**

```bash
python3 PE-001.py
```

**Rust**

```bash
cargo run --bin PE-001
```
